# Quickstart

Get started with Grok CLI in under 5 minutes.

## Installation

```bash
npm install -g grok-cli-hurry-mode
```

## Configuration

```bash
grok config set apiKey YOUR_API_KEY
```

## First Command

```bash
grok "Help me understand this project"
```
